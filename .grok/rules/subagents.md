# Subagents (always-on stub)

> Tools / skills / subagents **allowed** when they improve quality/speed **or cost less than primary**.
> Full: `.grok/rules/on-demand/subagents.md` · routing: `.grok/rules/agent-routing.md`
> Always-on: `.grok/rules/auto-minimize.md`

**Quick:**
- Easy work → cheap/free model or secondary agent
- Hard product/logic → primary agent owns
- Verify/gates → subagent OK **only if cheaper** than primary running the gate
- Graphify/vision → same cost/ROI test
